# Wonder4D

## 环境要求：
#### JDK：v1.8
#### node.js ： v12.10.0   (gnvm 版本管理)
#### Yarn 
#### Android Studio ：
#### Android SDK  ：Android 10 (Q)
#### Android SDK Platform 29
#### Intel x86 Atom_64 System Image
#### Android SDK Build-Tools ： React native 29.0.2
#### Virtual Device ：API Level 29 image
